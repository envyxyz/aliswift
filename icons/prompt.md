# Icons

AliSwift extension icons need to be added here before publishing to the Chrome Web Store.

## Required Icons

Create the following PNG files in this directory:

- **icon48.png** - 48x48 pixels (used in toolbar and menus)
- **icon128.png** - 128x128 pixels (used in Chrome Web Store)

## Design Guidelines

Both icons should be:
- **Color**: Orange (#ff4f00 for light mode)
- **Style**: Simple, bold design reflecting the lightning bolt/speed theme
- **Background**: Transparent or orange square
- **Format**: PNG with alpha channel

## Quick Generation

You can use any image editor (Photoshop, Figma, GIMP) or online PNG generator to create these files. Here's a quick approach:

1. Create a new 48x48 canvas with transparent background
2. Draw a lightning bolt in orange (#ff4f00)
3. Export as `icon48.png`
4. Create a new 128x128 canvas and repeat
5. Export as `icon128.png`

For quick prototyping, you can also use online tools like:
- https://www.favicon-generator.org/ 
- https://www.online-convert.com/

Alternatively, place any 48x48 and 128x128 PNG icons here temporarily for development testing.
